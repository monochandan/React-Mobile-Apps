# [Movie App with React Native & Expo](https://github.com/monochandan/React-Mobile-Apps/tree/main/movie-app)



A mobile app for discovering, searching, and viewing trending movies using TMDB API and Appwrite for backend services.

https://github.com/user-attachments/assets/3a892b6b-02b8-402c-b3b6-cc9b0724eae1



## 🛠️ Features

- **Expo Setup**: Built with Expo SDK for cross-platform development.
- **Routing & Navigation**: Uses Expo Router for file-based navigation. 
- **Custom App Icon & Title**: Customized via `app.json`.
- **TMDB API Integration**: Fetches movies, trending lists, and details. 
- **Custom Hooks**: Reusable logic for data fetching and state management.
- **Search Screen**: Real-time movie search with optimized queries. 
- **Appwrite Backend**: Tracks user searches to rank trending movies.
- **Trending by Search**: Popularity algorithm based on user behavior. 
- **Movie Details**: Full-screen view with ratings, cast, and synopsis.
- **UI Optimization**: Smooth rendering with `FlatList`, image caching, and loading states.
- **Status Bar**: Hidden for immersive experience using `expo-status-bar`


# [Real Estate with React Native & Expo](https://github.com/monochandan/React-Mobile-Apps/tree/main/real-Estate)


https://github.com/user-attachments/assets/c52d4f6c-e982-4f1a-8120-7c1846b5d96c


A full-stack real estate application built with React Native, Expo, Appwrite, and Nativewind, enabling users to browse, search, and explore properties seamlessly. 

🚀 Features
- **Authentication**: Google OAuth login via Appwrite.
- **Navigation**: Tab-based and dynamic screen routing using Expo Router.
- **UI Styling**: Tailwind CSS with Nativewind for responsive design. 
- **Fonts & Assets**: Custom fonts and icons setup.
- **Home Screen**: Displays latest and recommended properties. 
- **Explore Screen**: Filter and discover properties by type, price, location. 
- **Property Details**: Full view with images, pricing, features, and agent info. 
- **Profile Screen**: User profile management and settings. 
- **Database**: Appwrite-powered backend with seeded property data. 
- **Data Fetching**: Centralized useAppwrite custom hook for CRUD operations.
