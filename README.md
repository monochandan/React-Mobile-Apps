# [Movie App with React Native & Expo](https://github.com/monochandan/React-Mobile-Apps/tree/main/movie-app)



A mobile app for discovering, searching, and viewing trending movies using TMDB API and Appwrite for backend services.

## 🛠️ Features

- **Expo Setup**: Built with Expo SDK for cross-platform development.
- **Routing & Navigation**: Uses Expo Router for file-based navigation. 
- **Custom App Icon & Title**: Customized via `app.json`.
- **TMDB API Integration**: Fetches movies, trending lists, and details. 
- **Custom Hooks**: Reusable logic for data fetching and state management.
- **Search Screen**: Real-time movie search with optimized queries. 
- **Appwrite Backend**: Tracks user searches to rank trending movies.
- **Trending by Search**: Popularity algorithm based on user behavior. 
- **Movie Details**: Full-screen view with ratings, cast, and synopsis.
- **UI Optimization**: Smooth rendering with `FlatList`, image caching, and loading states.
- **Status Bar**: Hidden for immersive experience using `expo-status-bar`
