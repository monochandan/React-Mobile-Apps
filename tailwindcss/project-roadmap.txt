- layouts, flexbox (structuring UI)
[
    - css properties: display, position, overflow.
    - relative: move property relative to its normal position.
    - absolute: moves the property to its nearest parent.
    - fix: fix the element in the view port, so it does not scroll.
    - sticky: untill scrolled to the certain pont, it behaves normally.

    - display: block --> takes full width.
    - display: inline --> behave like text, no width and hght.
    - display: flex --> enable flex box
    - display: grid --> enable css grid property.
]
